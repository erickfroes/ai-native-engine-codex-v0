---
name: engine-bootstrap
description: Use esta skill quando a tarefa for iniciar, organizar ou alinhar a estrutura de um game engine AI-native com Codex, incluindo contratos, skills, agentes, schemas e ferramentas de validação.
---

Passos:

1. Confirme o escopo da versão atual.
2. Liste os contratos mínimos necessários.
3. Garanta AGENTS.md, config do Codex, subagentes e skills básicas.
4. Garanta schemas mínimos de cena, componente e rede.
5. Garanta esqueleto MCP para validação e benchmarking.
6. Produza um backlog curto e vertical.

Saída esperada:

- estrutura do repo coerente
- contratos mínimos
- fluxo operacional do Codex definido
- backlog inicial pronto para execução
